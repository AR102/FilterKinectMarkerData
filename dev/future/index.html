Known Problems

Tests On Windows

Currently, automated tests over GitHub Actions aren't run on Windows. This is because the default Windows image on GitHub Actions doesn't have the necessary display drivers to compile GLMakie.

On Linux, this was easily fixed by installing drivers through apt. Maybe winget could be used?

Scrolling Speed In Menu

The menu for selecting the marker to use is very hard to use as scrolling is extremely slow.

This is an issue in Makie.jl and should hopefully be resolved soon, see https://github.com/MakieOrg/Makie.jl/pull/2616.
